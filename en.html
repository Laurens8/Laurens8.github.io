API

Node.js API with Express

This Node.js API, powered by Express and Mongoose, is designed to streamline campus management for Thomas More School. It offers features like campus creation, deletion, and retrieval, as well as robust user authentication for administrators and users. With intuitive CRUD operations and Mongoose's interface for MongoDB, managing campus data becomes efficient and organized. The API prioritizes security with built-in authentication mechanisms, ensuring only authorized personnel can modify records. Whether you're an administrator or a user, this API provides a user-friendly solution for accessing information on Thomas More School's campuses.
